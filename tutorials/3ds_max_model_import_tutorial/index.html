Ripping Models with 3ds Max

A tutorial for importing models from both episodes of Sonic the Hedgehog 4.

Note: This tutorial only works with the model formats used in the PC, Wii, and Xbox 360 ports (ZNO, GNO, and ENO, respectively)

1. Running the MaxScript

Before you begin, you will need to run this MaxScript (written by RandomTBush). To do so, select "Scripting > Run Script...", then select the MaxScript.

2. Importing a model

When you run the script, this window should appear:

Select one of the options, then select the file you want to import.

(Note: As mentioned above, the MaxScript does not support the model formats used in the PlayStation 3 and Mobile ports)

3. Exporting a model

To export a model, select "File > Export > Export..." and save the model wherever you want (you can also change the file type before saving).

The Sonic 4 Bonus Site is not associated with SEGA, Sonic Team, or Dimps.
