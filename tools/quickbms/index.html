QuickBMS

QuickBMS is a program that can extract, import, or convert files using scripts.

QuickBMS

QuickBMS can be downloaded on GitHub or the official website.

Scripts

A list of QuickBMS scripts for Sonic the Hedgehog 4: Episodes I and II's formats.

  • vector.bms (Backup link) - This QuickBMS script can extract the OBB and DZ files from the mobile ports of Sonic the Hedgehog 4: Episode I. This script doesn't work with the second episode's OBB file because it's in a different format than the first episode's aforementioned archives (see the OBBExtract page).
  • sonic4.bms (Backup link) - This QuickBMS script can extract contents of the AMB files of the PC, Wii, PlayStation 3, and Xbox 360 versions of Sonic the Hedgehog 4: Episode I, and the PC, Mobile, PlayStation 3, and Xbox 360 versions of Sonic the Hedgehog 4: Episode II.
