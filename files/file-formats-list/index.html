File Formats

List of file formats used in each port of both episodes of Sonic the Hedgehog 4.

Sonic the Hedgehog 4: Episode I

Common/Shared

  • CPK - Archive file (Wii, PlayStation 3, Xbox 360)
  • AMB - Archive file
  • DDS - Image file
  • AAX - Audio file
  • ADX - Audio file
  • CSB - Audio archive (PC, Wii, PlayStation 3, Xbox 360)
  • MSG - Text file (Wii, PlayStation 3, Xbox 360)

Microsoft Windows

  • ZNO - Model file
  • ZNM - Animation file

Mobile

  • DZ - Archive file (OBB file is the same format)
  • PNG - Image file
  • PRV - Sprite/Texture file
  • INO - Model file
  • INM - Animation file
  • MP3 - Audio file
  • WAV - Audio file

Wii

  • GVR - Sprite/Texture file
  • GNO - Model file
  • GNM - Animation file
  • BRSAR - Audio archive

PlayStation 3

  • CNO - Model file
  • CNM - Animation file

Xbox 360

  • ENO - Model file
  • ENM - Animation file

Sonic the Hedgehog 4: Episode II

Common/Shared

  • CPK - Archive file (PlayStation 3, Xbox 360)
  • AMB - Archive file
  • DDS - Sprite/Texture file (Windows, Mobile, PlayStation 3, Xbox 360) (The mobile version's .DDS files are in a different format than the standard .DDS files)
  • AAX - Audio file
  • ADX - Audio file
  • CSB - Audio archive

Microsoft Windows

  • ZNO - Model file
  • ZNM - Animation file

Mobile

  • OBB - Android archive file
  • LNO - Model file
  • LNM - Animation file
  • WAV - Audio file
  • MP4 - Video or audio file (Music files are stored as MP4 files instead of WAV files)

PlayStation 3

  • CNO - Model file
  • CNM - Animation file

Xbox 360

  • ENO - Model file
  • ENM - Animation file

The Sonic 4 Bonus Site is not associated with SEGA, Sonic Team, or Dimps.
